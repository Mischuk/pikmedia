В стоимость работ включена бесплатная тех. поддержка сайта в объеме 3 часов, в рамках которой можно дорабатывать сайт.

В стоимость работ по оптимизации сайта под поисковые запросы входит: разработка seo оптимизированный текстов. Каждый текст не менее 1500 знаков.

Наименование работ, входящих в состав услуги поисковая оптимизация

  • Анализ сайта и его характеристик (домен, контент, структура).
  • Распределение запросов по страницам, составление схемы «страница-запрос» для сайта.
  • Составление технического задания для копирайтеров по написанию оптимизированных текстов для сайта.
  • Разработка текстов, добавление готовых текстов на сайт.
  • Добавление сайта на индексацию в поисковые системы, инициализация в сервисах для вебмастеров.
  • Установка систем аналитики для отслеживания статистики сайта. Счетчики Яндекс Метрика и Гугл Аналитикс
  • Подготовка и добавление в корневую директорию сайта специальных файлов sitemap.xml и robots.txt (для роботов поисковых систем).
  • Оптимизация кода страниц и разметки: мета-теги, title, теги выделений и заголовков.
  • Оптимизация рисунков и перелинковка внутренних страниц.
  • Техническая оптимизация сайта: исключение дублей сайта и отдельных страниц, простановка перенаправлений, приведение кода сайта в соответствие требованиям поисковых систем.
